Your new favorite operating system designed with speed, style, and ease of use.

Key Features

Modern User Interface

With a sleek, Windows XP-inspired theme, NovaXP offers an immersive and intuitive interface.

Draggable Windows

Enjoy the flexibility of dragging windows across your screen, just like the classic desktop experience.

Taskbar with App Management

Easily manage your open applications with a taskbar that supports app reopening and system notifications.

File Explorer

Explore and manage files effortlessly with a modern and responsive file manager integrated into the OS.

Installation Guide

Follow these steps to install NovaXP OS on your system:

  • Download the installer from the official GitHub page.
  • Run the setup and follow the on-screen instructions.
  • Choose your desired partition and wait for the installation to complete.
  • After installation, reboot and enjoy your new NovaXP experience!

System Requirements

  • CPU: 1 GHz or higher
  • RAM: 2 GB or higher
  • Storage: 5 GB free space
  • Graphics: Compatible with modern GPUs

Support

If you run into any issues with NovaXP OS, here are some solutions:

Troubleshooting

  • Installation Failed: Ensure your system meets the minimum requirements and try again.
  • File Explorer Not Responding: Restart the OS and try opening File Explorer again.
  • Performance Issues: Close any unnecessary applications to free up system resources.

For further assistance, reach out on our Support Forum.

Contribute

How to Contribute

  1. Fork the repository on GitHub.

  3. Submit a pull request with a clear description of your changes.

Contribution Guidelines

  • Ensure your code follows the existing style guide.
  • Test all new features thoroughly before submitting your pull request.
  • Respect the work of others and contribute constructively.
